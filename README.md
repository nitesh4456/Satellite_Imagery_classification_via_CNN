The Code Source contains an approach to build a CNN (Convolutional Neural Network) from scratch and to make it capable of identifying Land Use from the unknown Satellite Imagery.
The Funations are written in Python
We have used EuroSat RGB dataset to train this CNN model.
Dataset source : https://www.kaggle.com/datasets/waseemalastal/eurosat-rgb-dataset

Dataset Used :
  No. of Labels -> 10
  No. of Samples used (both for training and for testing as well ) -> 10000 (1K images per class)
